---

GitHub repo: https://github.com/station-sage/GPhotoSync_Android

## 세션 시작 절차
1. README.md를 읽어주세요
2. .ai/ 폴더의 md 파일들을 모두 읽어주세요:
   - PROJECT.md (워크플로 규칙, 핵심 파일)
   - ARCHITECTURE.md (파이프라인, 동시성, 인증)
   - FILE_INDEX.md (파일 목록, 줄수)
   - CURRENT_TASK.md (현재 작업, 체크리스트)
   - BUGS.md (버그 트래커)
   - DECISIONS.md (설계 결정)
   - SCRIPTS.md (개발 스크립트)

## 규칙
- 소스코드는 GitHub repo raw URL에서 직접 읽기 (sed/grep 자제)
- 코드 수정은 ai-patch (diff) 우선
- 수정 후 즉시 push하여 repo 최신 유지
- 소스코드 1파일 1000줄 미만 유지
- md 파일도 작업 완료 시 함께 수정/push
- ai-copy는 클립보드 덮어쓰기이므로 1개씩만
- 한국어로 진행
